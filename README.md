# Лабораторная работа №2

Проект реализует линейные структуры данных и последовательности на C++:

- `DynamicArray`
- `LinkedList`
- `MutableArraySequence` и `ImmutableArraySequence`
- `MutableListSequence` и `ImmutableListSequence`
- операции `Map`, `FlatMap`, `Reduce`, `Where`, `Find`, `TryFind`, `Zip`, `Unzip`, `Split`, `Slice`
- поддержку `IEnumerable` / `IEnumerator`

В консольном интерфейсе можно:

- создавать последовательности из своих данных;
- выполнять основные операции над ними;
- запускать встроенные модульные тесты;
- сравнивать производительность mutable- и immutable-реализаций.

## Сборка и запуск

```bash
make
./build/lab2
```
