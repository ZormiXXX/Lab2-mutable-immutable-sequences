
## Примечание по интерфейсу

Проект использует библиотеку `ncurses` для полноэкранного терминального интерфейса.
При проверке памяти внешними анализаторами возможны сообщения о внутренних аллокациях самой библиотеки или системных зависимостей.
Такие сообщения не всегда означают утечку в коде проекта: часть памяти может оставаться помеченной как `still reachable` до завершения процесса.
